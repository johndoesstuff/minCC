## Currently Supports:

- If Statements
- Functions
- While Loops
- For loops
- Variable Declaration / Assignment
  - `=`, `+=`, `-=`, `*=`, `/=`
  - Incrementing / Decrementing
- Dereferencing + Pointer Arithmatic
- Array indexing
- Types
  - `int`
  - `bool`
  - `char`
  - strings (`char*`)
    - escape codes `\n`, `\t`, `\0` (for `char` only)
  - `float`
  - `double`
  - `short`
  - `long`
  - Arbitrary pointer depth
  - Pass by reference
- Native Functions
  - printf
- Type Coercion
  - Variadic Function Promotion
- Scopes
- Error Messages
- Binary and Unary Operators
- Main Function
